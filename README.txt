國度321 YT上傳提醒 App

手機安裝方式
1. 先把整個 yt-reminder-app 資料夾放到支援 HTTPS 的網站空間。
2. iPhone：用 Safari 開啟網址，按「分享」，選「加入主畫面」。
3. Android：用 Chrome 開啟網址，按「安裝 App」或選單中的「加到主畫面」。
4. 在 App 內按「加入行事曆」，即可建立每天上午 9 點的系統提醒。

功能
- 今日、接下來、全部排程
- 完成勾選（保存在手機內）
- 儲存及開啟 YouTube 影片連結
- 可離線使用
- 可安裝到手機桌面
- 匯出手機行事曆提醒

重要說明
一般手機網頁無法在完全關閉後可靠地自行排程本機通知，因此本 App 以手機行事曆提醒作為主要提醒方式。
